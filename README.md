# Portfolio

Ahmad Faidzal Ibrahim's personal portfolio, built with [Astro](https://astro.build).

Content lives in `src/content/` (projects, experience, education, certifications,
gallery) and `src/data/` (about, skills, site settings). Editing will move to
Keystatic (Phase 1 steps 5-6, in progress). See `docs/adr/` for the decisions
behind the stack.

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
npm run preview  # serve the built site
```
